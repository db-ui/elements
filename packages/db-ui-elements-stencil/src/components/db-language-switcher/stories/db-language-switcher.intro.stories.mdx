<Meta title="07-Utility/Language Switcher/Intro" />

<db-headline variant="1">&lt;db-language-switcher&gt;</db-headline>

The `<db-language-switcher></db-language-switcher>` element represents list with languages.

There are three options to use &lt;db-language-switcher&gt;:

<db-headline>Language Switcher with custom `li` wrapper</db-headline>

<i>
  <strong>
    Maybe you need this option to use framework specific listeners.
  </strong>
</i>

<Canvas>
  <db-language-switcher aria-selected-index="1">
    <li>
      <db-link href="#">EN</db-link>
    </li>
    <li>
      <a href="#">DE</a>
    </li>
  </db-language-switcher>
</Canvas>

<db-headline>Language Switcher with &lt;db-link&gt;</db-headline>

<i>
  <strong>
    Use with caution, you won't have reactivity on changed HTML after the initial load.
  </strong>
</i>

<Canvas>
  <db-language-switcher aria-selected-index="1">
    <db-link title="English language" href="#">
      EN
    </db-link>
    <db-link title="Deutsche Sprache" href="#">
      DE
    </db-link>
  </db-language-switcher>
</Canvas>

<db-headline>Linklist with data</db-headline>

<Canvas>
  <db-language-switcher aria-selected-index="1" data={mockDataLanguages()} />
</Canvas>

The data should be an object passed in with `JSON.stringify()`

```typescript
export type DbLinkType = {
  href: string;
  title: string;
  label: string;
};
```
