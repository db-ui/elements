<db-headline variant="1">&lt;db-overflow-menu&gt;</db-headline>

The `<db-overflow-menu></db-overflow-menu>` element represents a overflow menu or a dropdown.

There are 3 options to use &lt;db-overflow-menu&gt;:

<db-headline variant="2">overflow-menu with wrappers</db-headline>

Use [&lt;li&gt; (**l**ist **i**tem)](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/li) as a wrapper tag around each element.

Only `<button>` and `<a>` HTML tags work as items content.

<i>
  <strong>
    Maybe you need this option to use framework specific listeners.
  </strong>
</i>
