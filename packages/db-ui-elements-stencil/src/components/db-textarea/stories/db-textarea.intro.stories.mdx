<db-headline variant="1">&lt;db-textarea&gt;</db-headline>

The textarea element represents a multiline plain text edit control. Most of attributes are used in the same way as with a native textarea. You can try out on <a href="/story/db-textarea--live-demo">Textarea Live Demo</a> Page.

<Canvas>
  <db-textarea label="Semitransparent" name="textarea-name"></db-textarea>
  <db-textarea label="White" name="textarea-name" variant="white"></db-textarea>
  <db-textarea
    label="Outline"
    name="textarea-name"
    variant="outline"
  ></db-textarea>
  <db-textarea label="Solid" name="textarea-name" variant="solid"></db-textarea>
</Canvas>

<db-headline>disabled</db-headline>

Make textareas look inactive by adding the disabled boolean attribute to any &lt;db-textarea&gt; element.

  <db-textarea label="Textarea" name="textarea-name" disabled></db-textarea>
</Canvas>
